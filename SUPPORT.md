## 日本語

### 用途別の窓口

| やりたいこと | 場所 |
| --- | --- |
| **使い方の質問・議論** | [GitHub Discussions](https://github.com/sserada/fuseji/discussions) |
| **バグ報告・改善要望** | [GitHub Issues](https://github.com/sserada/fuseji/issues/new/choose) |
| **セキュリティ脆弱性の非公開報告** | [SECURITY.md](SECURITY.md) を参照（GitHub Security Advisory もしくはメール） |
| **貢献方法・開発フロー** | [CONTRIBUTING.md](CONTRIBUTING.md) |
| **行動規範** | [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) |
| **ロードマップ** | [ROADMAP.md](ROADMAP.md) |
| **v0.2 → v0.3 アップグレード手順** | [MIGRATION.md](MIGRATION.md) |

### 質問する前に

- [README.md](README.md) と [docs/api.md](docs/api.md) のサンプルコードを試す
- [既存の Issue](https://github.com/sserada/fuseji/issues?q=is%3Aissue) を検索して同じ問題が報告されていないか確認
- [`examples/`](examples/) 配下の動作する最小サンプルを参照

### Issue を立てるときに含めると助かる情報

- fuseji のバージョン (`pip show fuseji` で確認)
- Python バージョン (`python --version`)
- OS (macOS / Linux / Windows + バージョン)
- 再現コードの最小例
- 期待する挙動と実際の挙動

### サポートの範囲

fuseji は OSS プロジェクトであり、商用サポート契約は提供していません。メンテナーはベストエフォートで対応します。応答 SLA はありませんが、**セキュリティ脆弱性の初動応答は 5 営業日以内** を目標としています（[SECURITY.md](SECURITY.md) 参照）。

## English

### Where to ask what

| Goal | Channel |
| --- | --- |

### Before opening an issue

- Try the samples in [README.en.md](README.en.md) and [`docs/api.md`](docs/api.md)
- Search [existing issues](https://github.com/sserada/fuseji/issues?q=is%3Aissue) for similar reports
- Check the runnable minimal samples under [`examples/`](examples/)

### Useful information for issues

- fuseji version (`pip show fuseji`)
- Python version (`python --version`)
- OS (macOS / Linux / Windows + version)
- Minimal reproducing snippet
- Expected vs actual behaviour

### Scope of support

fuseji is an open-source project with no commercial support contract. Maintainers respond on a best-effort basis with no response-time SLA, except that **security vulnerabilities target a first response within 5 business days** (see [SECURITY.md](SECURITY.md)).
